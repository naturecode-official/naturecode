# NatureCode - Cross-platform Terminal AI Assistant

## Overview

NatureCode is a professional-grade terminal AI assistant designed for developers and advanced users. It provides intelligent code assistance, project management, and AI-powered development tools directly from your terminal.

## Version: 1.4.5.4

### Key Features

- **Multi-AI Provider Support**: DeepSeek, OpenAI, and Ollama integration
- **Professional Installation**: Automated, non-interactive installation system
- **Code Intelligence**: Static analysis, refactoring suggestions, quality metrics
- **Git Integration**: Commit message generation, change analysis, branch management
- **Project Management**: Dependency analysis, security scanning, performance monitoring
- **Plugin System**: Extensible architecture with custom plugin support
- **Session Management**: Save, restore, and manage multiple AI sessions
- **Cross-platform**: Works on macOS, Linux, and Windows

## Quick Installation

### One-line Install (Recommended)

```bash
curl -fsSL https://raw.githubusercontent.com/naturecode-official/naturecode/main/install.sh | bash
```

### Manual Installation

```bash
# Clone repository
git clone https://github.com/naturecode-official/naturecode.git
cd naturecode

# Install globally
npm install -g .
```

## Getting Started

1. **Configure AI Model**:

   ```bash
   naturecode model
   ```

2. **Start Interactive Session**:

   ```bash
   naturecode start
   ```

3. **Get AI Assistance**:

   ```bash
   naturecode help "your question"
   ```

4. **Check Available Commands**:
   ```bash
   naturecode --help
   ```
